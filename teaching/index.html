Resources

  • Moon Balls Lunar Phases Activity
    A hands-on activity to teach the phases of the moon, developed for the IYSF program and based on activities from the UAYF. Suitable for 4th grade and beyond.

  • LaTeX Examples with Overleaf
    Introductory lesson on using LaTeX in Overleaf
    • Link to GitHub repository
  • A Beginner's Guide to Working with Astronomical Data
    A wonderful introductory guide by Markus Pössel to help students get started with astronomical research and tools.
